# hr_dashboard-
Developed an interactive Power BI dashboard to analyze employee performance and training data. Performed data cleaning and transformation, including handling missing values, converting categorical performance scores to numeric metrics, and removing unnecessary columns. Designed KPIs and visualizations to track employee performance, training outcomes, engagement, and training costs. Built multi-page dashboards with slicers and filters to enable dynamic exploration of HR insights and support data-driven decision making.
